# Shopee Cookie Scraper Extension

Chrome extension for extracting Shopee cookies to use with the Affiliate Dashboard.

## Installation

1. Extract this ZIP file
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the extracted folder

## Usage

1. Login to Shopee (shopee.co.id, shopee.com.my, etc.)
2. Click the extension icon in Chrome toolbar
3. Click "Extract Shopee Cookies"
4. Click "Send to Dashboard"
5. Cookies will be sent to your local dashboard (localhost:8000)

## Note

This extension only works when your dashboard backend is running on localhost:8000.
